    # C Quick IDE

A lightweight C IDE for Ubuntu built using PyQt5.

## Features

- Create / Open / Save C files
- Compile using gcc
- Run using ./a.out
- Built-in output panel
- Keyboard shortcuts (Ctrl+S, Ctrl+O, F5)
- Dark theme
- Toolbar with icons

## Installation

sudo apt install python3-pyqt5 gcc
python3 main.py
